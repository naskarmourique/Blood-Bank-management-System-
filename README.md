# BloodConnect: National Blood Management System

![BloodConnect Logo](favicon.jpg)

**BloodConnect** is a full-stack web application designed as a **National Blood Management System**. It connects **blood donors, patients, and blood banks**, streamlining donation, inventory management, and urgent requests to save lives.

## ðŸš€ Features

- **Interactive Dashboard**: Real-time blood inventory, quick-action cards, recent activity logs, and pending emergency requests.  
- **Donor Registration**: Easy-to-use form with client-side validation for individuals & organizations.  
- **Urgent Blood Requests**: Post emergency requests with patient and hospital details.  
- **Blood Drive Events**: View, filter, and request new donation events.  
- **Services & Feedback**: Informational page with image carousel and feedback form.  
- **Contact & Support**: Emergency helpline, multiple support channels, and embedded map.  
- **Modern UI**: Bootstrap 5 + custom CSS, glassmorphism, smooth animations, fully responsive.  
- **Secure Backend**: PHP + MySQL to handle all form submissions safely and manage data efficiently.
